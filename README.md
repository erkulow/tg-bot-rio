# 🚛 Truck List Bot

Telegram бот для мониторинга статуса **READY** в Excel файле Truck List.

---

## ⚙️ Установка

```bash
npm install
```

---

## 🔧 Настройка (bot.js — первые 6 строк)

```js
const TOKEN   = "YOUR_BOT_TOKEN_HERE";  // токен от @BotFather
const CHAT_ID = "YOUR_CHAT_ID_HERE";    // ID чата/группы
const EXCEL_FILE = path.resolve(__dirname, "truck_list.xlsx"); // имя файла
```

### Как получить TOKEN:
1. Напишите @BotFather в Telegram
2. Отправьте `/newbot`
3. Скопируйте выданный токен

### Как получить CHAT_ID:
1. Запустите бота
2. Напишите боту `/chatid`
3. Скопируйте число в ответе

---

## 🚀 Запуск

```bash
# Положите Excel файл рядом с bot.js
cp /путь/к/truck_list.xlsx ./truck_list.xlsx

# Запустить
npm start
```

---

## 📋 Команды бота

| Команда   | Описание                          |
|-----------|-----------------------------------|
| `/start`  | Приветствие и список команд       |
| `/check`  | Ручная проверка прямо сейчас      |
| `/status` | Статус бота и путь к файлу        |
| `/chatid` | Узнать ID текущего чата           |

---

## 🔄 Как работает

1. При запуске сразу делает первую проверку
2. Каждые **5 минут** читает Excel файл
3. Ищет ячейки со значением `"Ready"` (регистр не важен)
4. Отправляет в Telegram: компанию, диспетчера, водителя, телефон, дату
5. Если READY не найдено — тоже отправляет об этом сообщение
